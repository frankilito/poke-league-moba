# 宝可梦联盟 · Poké League (3D MOBA)

一款完整可玩的宝可梦主题 3D《英雄联盟》类 MOBA 游戏,支持**双人在线联机**。
纯本地运行:Three.js 渲染 + Pokemon-3D-api 的真·3D 宝可梦模型(Draco压缩GLB) + PokeAPI 官方立绘/道具图标/叫声,素材全部已本地化。

**🎮 在线试玩: https://frankilito.github.io/poke-league-moba/**

## 启动

- 在线:直接打开上面的链接
- 本地:双击 `启动.command`(自动起本地服务器并打开浏览器)

## 双人联机(2人协力 VS 电脑)

1. 玩家A 在选人界面点「创建房间」,把 4 位房间码发给好友
2. 玩家B 输入房间码点「加入」
3. 双方各选一只宝可梦,房主点「出发对战」
4. 两人同在蓝方(A+B+3个AI队友) VS 5个电脑,摧毁敌方大师球获胜

技术:WebRTC 点对点(PeerJS 云信令),主机权威模拟 + 10Hz 状态快照同步,客机木偶世界+输入转发。双方浏览器直连,无需游戏服务器。

## 玩法

- **5V5 三路推塔**:摧毁敌方「大师球基地」获胜。防御塔是巨型精灵球,需按 外塔→高地塔→基地塔→大师球 顺序推进
- **12 位英雄**:皮卡丘/喷火龙/水箭龟/妙蛙花/耿鬼/胡地/怪力/拉普拉斯/卡比兽/快龙/超梦/九尾,每位 4 个原作招式改编技能(QWER),附带全套 3D 特效(天雷/光束/暴风雪/龙卷风/护盾泡/冲击波…)
- **大招演出**:每个技能都有过场动画——普通技能(QWE)触发紧凑快速立绘横幅,大招(R)触发半屏立绘大过场(斜切面板+速度线+官方立绘冲入);大招命中附带"炸裂级"冲击反馈——冲天光柱/地面裂纹/碎石飞溅/放射速度线/双层冲击波/全屏闪光/镜头震动/打击顿帧
- **施法目标逻辑**:左键点击敌人锁定目标(红圈标记);施法优先级为 鼠标悬停 > 锁定目标 > 攻击目标 > 范围内最近英雄 > 最近敌人,弹道技能自动预判走位;位移技能(闪现/冲刺)始终朝鼠标方向
- **小兵**:小拉达(近战)/波波(远程)/小磁怪(炮车),每 32 秒一波
- **野区**:椰蛋树长老(蓝buff)/隆隆岩长老(红buff)/狂暴暴鲤龙(团队伤害加成)/远古大岩蛇(男爵级团队buff)
- **经济系统**:补刀/击杀/助攻/推塔金币,泉水商店 24 件宝可梦道具(讲究头带/生命宝珠/吃剩的东西/神奇糖果/气势披带…)
- **成长**:1-18 级,技能加点(Ctrl+QWER 或点技能上方 +),经验共享
- **宝可梦世界生态场景**:高精度程序化3D模型(团状树冠橡树/樱花树+飘落花瓣/多层松树/碎面巨岩,顶点抖动+面片着色);高草丛/花海/蘑菇/莓果树/散落的精灵球道具;河道活水+荷叶+鹅卵石+两座木桥+中央石板广场;野怪营地悬浮水晶、龙坑/男爵坑岩石巢穴;基地队旗/泉水池/石灯笼/围栏;天上有流云和飞翔的波波;野区有游荡的走路草/伊布/呆呆兽/巴大蝶,河里的鲤鱼王会定时跃出水面;夜光萤火虫点缀丛林
- **电脑角色**:两队均为有完整行为逻辑的 AI(对线/推进/风筝/撤退/回城购物/技能预判);顶栏「➕添加AI」可随时给任意一方增派新的电脑英雄(最多每队9人)
- **难度**:简单/普通/困难(影响AI反应速度/命中预判/技能率)

## 操作

| 键位 | 功能 |
|---|---|
| 右键 | 移动 / 攻击目标 |
| 左键点敌人 | 锁定技能目标(点空地取消) |
| Q W E R | 释放技能(优先打锁定/悬停目标) |
| Ctrl+Q/W/E/R | 升级技能 |
| D / F | 伤药喷雾 / 空间转移(闪现) |
| A+左键 | 攻击移动 · S 停止 |
| B | 回城 · P 商店 · Tab 战绩 |
| Y | 镜头锁定切换 · 空格 看向英雄 · 滚轮 缩放 · Esc 菜单 |

## 技术

- `js/data.js` 英雄/技能/装备/野怪数值
- `js/effects.js` 粒子/光束/闪电/领域 3D 特效引擎(程序化纹理)
- `js/entities.js` 单位/英雄/小兵/精灵球塔/大师球基地
- `js/scenery.js` 场景生态(植被/河道/营地/基地装饰/云鸟/野生宝可梦)
- `js/ai.js` 电脑行为状态机
- `js/game.js` 地图/技能模式/经济/输入/主循环
- `js/ui.js` 图鉴风 HUD/商店/计分板/小地图/播报
- 模型: `assets/models/*.glb` (Draco) · 立绘: `assets/art` · 叫声: `assets/cries` · 道具: `assets/items`
